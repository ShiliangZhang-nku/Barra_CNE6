# Barra_CNE6

# 代码结构：

1.barra_template.py: 

实现功能：

1）对文件夹内csv文件读取添加一层封装，以实现通过访问因子类的属性即可读取因子矩阵数据；

2）实现因子名称的模糊匹配并忽略其大小写；


2.barra_CNE6_factor.py:

实现功能：

使用dask库，对原始矩阵数据进行批量并行计算，生成CNE6日频因子数据。
